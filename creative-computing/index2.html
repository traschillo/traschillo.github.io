Hints

  • Use the CSS rgb() notation to set the color. See documentation here.
  • We can create percentages by dividing an input value by the highest value it could ever reach. For example, the mouse X position can never be greater than the width of the page, so to turn it in to a percentage we would do currentMouseX / widthOfPage. Once we have a percentage, we can multiply it by our target maximum value. For example, the maximum value of a color channel with rgb() is 255.
  • You can get the dimensions of the page with window.innerWidth and window.innerHeight.
  • Note that rgb() requires whole numbers. If you have decimal places in a number, you can remove them by using Math.round(). See documentation here.
